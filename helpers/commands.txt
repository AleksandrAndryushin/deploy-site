Здесь будут разные консоль команды ->

Для того что бы начать работать в проекте
npm i

Для того что бы запустить сервер
npm run dev

Для того что бы остановить сервер
ctrl + c

Инструкция как правильно коммитить на репозиторий
1 - git rm -r --cached .
2 - git add .
3 - git commit -m "Описание"
4 - git push
